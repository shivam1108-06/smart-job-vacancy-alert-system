# Smart Job Vacancy Alert System

A modern full-stack Job Vacancy Alert System built with React.js, Node.js, Express.js, PostgreSQL, and Prisma.

## 🚀 Features

- User Authentication
- Job Search
- Advanced Filters
- Save Jobs
- Apply for Jobs
- Email Notifications
- Admin Dashboard

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL
- Prisma ORM
